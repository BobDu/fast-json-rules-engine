# fast-json-rules-engine

> Same rules JSON, compiled once — no promises, no clones.

Compiled, synchronous, zero-dependency evaluator compatible with the
[json-rules-engine](https://github.com/CacheControl/json-rules-engine) JSON rule
format, built for workloads where facts are plain static values.

**Status: work in progress — not yet published.**

This is an independent project, not affiliated with json-rules-engine.
